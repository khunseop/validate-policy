# 웹 애플리케이션 사용 가이드

## 개요

방화벽 정책 검증 시스템을 웹으로 실행할 수 있습니다.  
Running/Candidate 정책 파일을 업로드하고 대상 정책으로 검증한 뒤, 결과 요약과 리포트를 확인·다운로드할 수 있습니다.

## 설치 및 실행

```bash
pip install -r requirements.txt
cd web
python app.py
```

브라우저에서 `http://127.0.0.1:5000` 접속

## 사용 방법

1. **벤더 선택**: Paloalto / SECUI
2. **정책 파일 업로드**
   - Running 정책 파일 (필수)
   - Candidate 정책 파일 (Paloalto 필수, SECUI는 같은 파일 사용 시 생략 가능)
   - 대상 정책 파일 (선택, 여러 개 가능)
3. **검증 시작** 후 결과 확인 및 리포트 다운로드

## 기능

- 파일 업로드 (드래그 앤 드롭 지원)
- SECUI: 같은 파일·별도 파일 모두 지원, 시트 목록 버튼으로 시트 선택
- 검증 결과 요약 표시
- 리포트 다운로드 (파일명에 날짜·시간 포함, 중복 방지)

## 제한

- 업로드 파일 최대 100MB
- Excel 형식(.xlsx, .xls)만 지원
